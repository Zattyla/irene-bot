# 🐰 IRENE.SYS
> A K-Pop & Utility Assistant for Discord, developed by Katty.

## 📌 About The Project
Irene is a state-of-the-art, custom-built Discord bot designed exclusively to manage, entertain, and protect K-Pop communities. From automated SNS tracking to advanced moderation, Irene keeps the server engaged and strictly organized.

---

## ✨ Core Features

### 📡 Global Radar (YouTube & Reddit)
Automatic 24/7 monitoring of top K-Pop agencies and artists. Irene fetches the latest MVs, teasers, and global news directly to the server with an intelligent anti-duplication system and a customized ping cooldown.

### 🛡️ Absolute Security & Auto-Mod
* **Smart Anti-Spam:** Automatically deletes suspicious links and server invites.
* **Banned List Protocol:** A strict, zero-tolerance database for problematic figures. Irene scans all fetched news and immediately blocks any mention of blacklisted individuals.

### ⏱️ Comeback Countdown & Calendar
* **Live Countdowns:** Tracks upcoming releases, adjusting automatically to each user's local timezone.
* **Birthday System:** Real-time KST (Korean Standard Time) automated birthday announcements for registered idols.

### 🎙️ AI Studio Integration
Integrated directly with **Katty's AI Studio**. Mentioning specific keywords automatically deploys the official Google Colab link, allowing users to create AI covers (RVC), isolate stems (UVR), and mix vocals with custom effects.

---

## 💻 System Commands

### 🌟 Public Commands
* `/info` - View official details, members, and links for a monitored group.
* `/monitored` - View the complete list of tracked artists and agencies.
* `/comebacks` - Check the countdown clock for upcoming releases.
* `/choice` - Let Irene randomly decide between multiple options.
* `/suggestion` - Send feedback or feature requests directly to the developer's DM.

### 🔒 Admin Protocols
* `/wake-up` - Manual override to restart tracking loops.
* `/add_group` / `/remove_group` - Manage the SNS Radar database.
* `/role_panel` - Deploy interactive button panels for automated roles.
* `/add_tobannedlist` / `/see_bannedlist` - Manage the server's security filters.
* `/update` - Announce new bot features with aesthetic patch notes.
